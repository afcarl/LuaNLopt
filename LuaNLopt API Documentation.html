1

Abstract

1.1 This is a projection of the NLopt C API as documented in ab-initio.mit.edu/.../NLopt_Reference to Lua.
1.2 Author: Rochus Keller, rkeller@nmr.ch
1.3 Issue: 2012-10-14

2

Definitions

2.1 nlopt
2.1.1 This is the module object generated by the statement require "LuaNLopt".
2.2 nlopt_opt
2.2.1 This is an object of the type nlopt_opt generated by nlopt.create or nlopt_opt:copy().
2.2.2 Note the ":" syntax in contrast to "."
2.3 nlopt.algorithm
2.3.1 Arguments of this type are integers which are members of the enumeration nlopt.algorithm.
2.3.2 The elements NLOPT_GN_DIRECT etc. of the enumeration of the C API are mapped to nlopt.algorithm.GN_DIRECT etc.
2.4 any
2.4.1 is any valid Lua type
2.5 array
2.5.1 is a Lua table used as an array of double; remember indices in Lua start with 1, not with 0!
2.6 nlopt.result
2.6.1 Arguments of this type are integers which are members of the enumeration nlopt.result.
2.6.2 The elements NLOPT_FAILURE etc. of the enumeration of the C API are mapped to nlopt.result.FAILURE etc.
2.7 n
2.7.1 The number of dimensions passed to nlopt.create
